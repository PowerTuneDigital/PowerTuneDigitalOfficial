Flash the image to your SD card ( you can user Etcher or do it via command Line ) Etcher can be downloaded her \href{https://etcher.io/}{\texttt{ https\+://etcher.\+io/}}

The image currently has a boot time of approx 17 seconds until Power\+Tune is starting ( i am still working on optimization )

About the image \+: The username and password are \+:

user\+: pi pw\+: raspberry

Not all linux commands are supported and there is no need to use sudo

\begin{DoxyItemize}

\item Log in with user and password from above
\item Switch to root user {\ttfamily \$ sudo su root}
\item Type the following\+:
\end{DoxyItemize}

{\ttfamily \$ wpa\+\_\+passphrase \char`\"{}\+Your\+SSIDHere\char`\"{} \char`\"{}\+Your\+Password\+Here\char`\"{} \texorpdfstring{$>$}{>}\texorpdfstring{$>$}{>} /etc/wpa\+\_\+supplicant/wpa\+\_\+supplicant.conf}


\begin{DoxyItemize}
\item Replace your WLAN SSID and Password Above
\end{DoxyItemize}

{\itshape Pro-\/tip\+: edit the file and delete the line\+: \#psk=Your\+Password\+Here}

{\ttfamily \$ reboot}

Now your pi will always log into WLAN at boot
